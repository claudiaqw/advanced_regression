En el dataset existen muchas filas que tienen valores Nan en alguna de las variables. En el caso de las variables , cerca de un 70% de los datos son descinocidos. Por esta razn no podemos eliminarlas. Por lo tanto, se recurre a la imputacin de valores. Para la imputacim se toma la media de los valores que ha tomado esa variable en ese mes y ao en ese estado especfico. Mediante la aplicacin de este mtodo se reduce el ruido introducido por los mtodos tradicionales de imputar con zeros o con algunos de los estadsticos de forma.

ampute <- function(df, index, state, date){
  new.df <- df %>% 
    filter(Province_State == state, month(Date) == month(date), year(Date) == year(date))
  value <- mean(new.df[,index], na.rm = TRUE)
  return(value)
}

data$Recovered[is.na(data$Recovered)] = 
  ampute(data, 8, data$Province_State, data$Date)

data$Total_Test_Results[is.na(data$Total_Test_Results)] = 
  ampute(data, 12, data$Province_State, data$Date)

data$People_Hospitalized[is.na(data$People_Hospitalized)] =
  ampute(data, 13, data$Province_State, data$Date)

data$Case_Fatality_Ratio[is.na(data$Case_Fatality_Ratio)] =
  ampute(data, 14, data$Province_State, data$Date)

data$Hospitalization_Rate[is.na(data$Hospitalization_Rate)] =
  ampute(data, 18, data$Province_State, data$Date)

data$People_Tested[is.na(data$People_Tested)] =
  ampute(data, 20, data$Province_State, data$Date)

data$Mortality_Rate[is.na(data$Mortality_Rate)] =
  ampute(data, 21, data$Province_State, data$Date)
